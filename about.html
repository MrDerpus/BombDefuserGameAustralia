About the program ...

I have created an easily customizable Bomb defuser mini game, to help immerse
players on the field or in Close Quarter Combat situations.

Written in C++ for speed & perfromance, to run seamlessly and effortlessly on even
the oldest and slowest computer or laptop.






Minimum system requirements:

Intel Xeon Processor X5650 CPU
Windows 7 32bit OS
50gb HardDrive space
4gb RAM
